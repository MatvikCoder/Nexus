# Nexus
Моя Римская Империя если проще

Компиляция и запуск просты: копируем репозиторий и в папке с ним запускаем compile.sh

Должно быть это так:

$ git clone https://github.com/MatvikCoder/Nexus.git

затем

$ cd Nexus

и, если у вас нету qemu, скачайте его.

Как только установили, пишите:

./compile.sh
Если возникла ошибка при запуске компиляции, напишите:
chmod +x compile.sh

а затем запустите компиляцию снова.


Разберём ситуацию с железом.

Что бы запустить нашу ОС на реальном железе, как не удивительно, нужен образ диска.

Просто так бинарник os.bin засунуть в великий вентой нельзя, поэтому у нас есть вариант создания образа дискеты, а для iso мы будем эмулировать нашу же дискету.


Если вы хотите запустить ОС на реальном железе, я приготовил для вас скрипт сборки iso!

Для него вам нужна поддержка genisoimage и dd.

Что бы всё правильно заработало, соберите нашу ос той же командой ./compile.sh.
Это нужно т.к. если вы вносили изменения в код и хотите что бы они работали на реальном железе, а не только в qemu.
Так же нужно скомпилировать снова (в особенности скомпилировать надо если вы не разу не запускали ОС) из за того, что iso образ собирается именно из бинарников boot.bin и kernel.bin

Как только вы сделали образ, поздравляю! Для запуска достаточно положить его на флешку с вентой, и в меню выбора для запуска выбираете загрузку в оперативную память.
