# Nexus
простая ОС на ассемблере nasm, запуск на эмуляторе qemu

Компиляция и запуск просты: копируем репозиторий и в папке с ним запускаем compile.sh

Должно быть это так:

$ git clone https://github.com/MatvikCoder/Nexus.git

затем

$ cd Nexus

и, если у вас нету qemu, скачайте его

Как только установили, пишите:

./compile.sh
